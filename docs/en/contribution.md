# Contribute

## This content is in progress

The Roadmap item you clicked is still being written. AI x Web3 School is gradually turning the roadmap into practical learning material across AI, Web3, agents, onchain automation, wallets, payments, and protocol coordination.

If you have experience with any of these topics, contributions are welcome.

## What you can contribute

- Write missing Roadmap entries
- Fix incorrect, unclear, or outdated explanations
- Add case studies, diagrams, code examples, or practice tasks
- Improve Chinese and English translations
- Share resources that help AI x Web3 builders learn faster

## How to submit a PR

1. Fork the repository: <https://github.com/lxdao-official/aiweb3school>
2. Create a new local branch.
3. Edit or add documents under `docs/zh/` or `docs/en/`.
4. Run the build check locally:

```bash
bun run docs:build
```

5. Open a Pull Request and describe what you added and which Roadmap topic it maps to.

If you want to discuss a topic before writing, opening an Issue is also fine.
