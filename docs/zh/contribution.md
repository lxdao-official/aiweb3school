# 参与共建

## 内容正在编写中

你点击的 Roadmap 条目还在整理和编写中。AI x Web3 School 会持续把 AI、Web3、Agent、链上自动化、钱包、支付和协议协作相关内容补齐成可学习、可实践的文档。

如果你对某个主题有经验，欢迎一起共建。

## 可以贡献什么

- 补充 Roadmap 中还没有完成的条目
- 修正文档中的错误、不准确表述或过时信息
- 增加案例、图解、代码示例和实践任务
- 帮助完善中英文翻译
- 提交你认为适合 AI x Web3 builders 学习的资源

## 如何提交 PR

1. Fork 仓库：<https://github.com/lxdao-official/aiweb3school>
2. 在本地创建新的分支。
3. 修改或新增 `docs/zh/`、`docs/en/` 下的文档。
4. 本地运行构建检查：

```bash
bun run docs:build
```

5. 提交 Pull Request，并说明你补充了哪些内容、对应 Roadmap 的哪个主题。

如果你只是想先讨论选题，也可以先开 Issue。
